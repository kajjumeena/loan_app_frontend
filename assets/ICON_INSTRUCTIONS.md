# LoanSnap Icon Assets

The image generation service is temporarily unavailable. Please create the app icons manually:

## Option 1: Use an Online Icon Generator

1. Go to https://www.appicon.co/ or https://icon.kitchen/
2. Upload a simple logo or use their templates
3. Choose these settings:
   - Background color: #6B46C1 (purple)
   - Icon: Lightning bolt or "L" letter
   - Style: Flat, modern
4. Download the generated assets
5. Replace the files in `assets/` folder

## Option 2: Use Figma/Canva

1. Create a 1024x1024px square
2. Add purple gradient background (#6B46C1 to #805AD5)
3. Add white lightning bolt or "LS" text
4. Export as PNG

## Required Files:

- `assets/icon.png` - 1024x1024px (app icon)
- `assets/splash.png` - 1242x2436px (splash screen)
- `assets/adaptive-icon.png` - 1024x1024px (Android adaptive icon)
